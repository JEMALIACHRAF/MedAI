Capabilities

Three Modules,
One Platform

Each module targets a specific clinical AI task — from binary lesion classification to retrieval-augmented medical Q&A.

🔬
Module 01
Lesion Classification
CNN trained on the HAM10000 dermoscopy dataset performs binary classification of skin lesions with confidence scoring.
  • Benign / Malignant detection
  • Confidence percentage output
  • 128×128 CNN inference
  • Sub-second latency on CPU
🧠
Module 02
Multimodal Analysis
GPT-4o Vision interprets clinical images with free-form clinical questions. Zero-shot classification for symptom text.
  • GPT-4o Vision (high detail)
  • Clinical VQA interface
  • Zero-shot text classification
  • 9-label pathology taxonomy
📚
Module 03
Clinical Knowledge RAG
Retrieval-augmented generation over curated medical documents — grounded, source-cited answers to clinical queries.
  • FAISS semantic retrieval
  • all-MiniLM-L6-v2 embeddings
  • GPT-3.5 generation
  • Source relevance scoring
⚠  This platform is intended for research and educational purposes only. Outputs do not constitute clinical diagnoses. Always consult a licensed healthcare professional.
Architecture

How It Works

A request flows through four stages — from user input to a grounded, explainable clinical output.

🖼
Input
Image or clinical text uploaded via drag-and-drop interface
⚙
Routing
FastAPI dispatches to the appropriate AI module endpoint
🤖
Inference
CNN, GPT-4o Vision, or RAG pipeline processes the query
📊
Output
Prediction, confidence score and sources returned in real-time
